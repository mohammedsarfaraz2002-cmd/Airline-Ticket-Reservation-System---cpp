# 🚆 Airline Ticket Reservation System (C++)

## 📌 Overview
This project is a console-based Airline Ticket Reservation System developed in C++. It allows users to book seats based on ticket class, view seating arrangements, and calculate the total cost of their bookings.

The system simulates a real-world seat reservation process with validation, seat availability checks, and a structured seating layout.

---

## 🎯 Features
- Seat reservation for 3 classes:
  - First Class (Rows 1–2)
  - Business Class (Rows 3–5)
  - Economy Class (Rows 6–13)
- Interactive seat selection (A–F per row)
- Real-time seat availability checking
- Seating chart display
- Booking summary with total cost calculation
- Input validation for ticket type and seat selection

## ▶️ How to Run
1. Open the project in Visual Studio (or any C++ IDE)
2. Compile the program
3. Run the executable
4. Follow on-screen instructions:
   - Select ticket class (F / B / E)
   - Choose row and seat
   - Continue booking or print seating chart
